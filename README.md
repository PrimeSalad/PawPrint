## Setup Instructions

Follow these steps to get the project running:

### Step 1: Install Dependencies
Install all required Python packages listed in `requirements.txt` using powershell on your vscode:

```bash
pip install -r requirements.txt
```

### Step 2: Run the python server
To run the Python server, use `serve.py` in PowerShell.

```bash
python serve.py
```
This will automatically start the live server. Once the webpage loads, you can use the dog breed prediction and report generation services.
